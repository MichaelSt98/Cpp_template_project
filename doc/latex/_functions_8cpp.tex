By default, non-\/pointer arguments in C++ are passed by value. When an argument is {\bfseries{passed by value}}, the argument\textquotesingle{}s value is copied into the value of the corresponding function parameter. Therefore the original argument can not be modified by the function!\hypertarget{_functions_8h_autotoc_md103}{}\doxyparagraph{Pros}\label{_functions_8h_autotoc_md103}
